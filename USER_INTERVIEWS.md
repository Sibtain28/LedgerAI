# User Interviews

## Interview 1: CTO of a Mid-sized Startup
**Pain Point**: "We have 4 different teams buying AI tools independently. I have no idea if we are double-paying for GitHub Copilot and Cursor simultaneously."
**Feedback on LedgerAI**: Loved the cross-tool redundancy detection. Suggested adding a specific rule for enterprise minimums (e.g., minimum 50 seats for some enterprise API tiers).

## Interview 2: VP of Engineering
**Pain Point**: "Our developers just use direct APIs with their personal cards and expense them. We have zero visibility into token leakage."
**Feedback on LedgerAI**: The ability to identify when API spend exceeds flat-rate tier boundaries was highly valued. Suggested integrating directly with Okta/SSO for automatic tool discovery in the future.
